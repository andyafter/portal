# Portal

This repo contains a minimal Next.js interface for playing heads-up poker with the AI server described in the docs above.

The client lives under `poker-table/` and assumes blinds of 1/2.

## Getting started

```bash
cd poker-table
npm install
npm run dev
```

The UI expects the server running at `http://127.0.0.1:8888`.

Run `npm run build` to create a production build.
